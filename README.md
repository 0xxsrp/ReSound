## 📖 English

**ReSound** is a desktop soundboard application built with Electron. It lets you play sounds through your speakers while simultaneously routing them to a virtual audio cable — so you can hear them yourself AND broadcast them through Discord voice chat.

### ✨ Features

| Feature | Description |
|---------|-------------|
| **Dual Audio Output** | Play through speakers + virtual cable at the same time |
| **Global Hotkeys** | Assign keyboard shortcuts to any sound (Ctrl, Shift, Alt + key) |
| **Play Queue** | Build a playlist and let sounds auto-play sequentially |
| **Favorites** | Star your most-used sounds and filter by favorites |
| **Multi-Select** | Select multiple sounds — batch play, delete, move, or clear hotkeys |
| **Categories** | Organize sounds into folders, create, rename, delete |
| **Drag & Drop** | Upload files directly or drag cards between categories |
| **Volume Control** | Per-sound volume, per-guild volume, and global volume |
| **Search** | Search by name or prefix `hk:` to filter by hotkey |
| **Sorting** | Name, newest, oldest, and hotkey priority |
| **Play Count** | Track how many times each sound has been played |
| **Keyboard Navigation** | Arrow keys, Home/End, Enter/Space |
| **Density Modes** | Compact, Medium, Spacious grid layouts |
| **Dark / Light Theme** | Catppuccin Mocha (dark) and Latte (light) |
| **Auto-Update** | Checks GitHub Releases for new versions |
| **Minimize to Tray** | Background operation with tray menu |

### 🚀 Installation

Download the latest installer from the **[Releases](https://github.com/0xxsrp/ReSound/releases)** page.

**Requirements:**
- Windows 10 or later
- [VB-Cable](https://vb-audio.com/Cable/) or any virtual audio cable (for mic routing)

### 🛠 Development

```bash
npm install        # Install dependencies
npm run start      # Launch the app
npm run build      # Build installer (NSIS)
npm run publish    # Build + upload to GitHub Releases
```

## 📖 العربية

**ReSound** هو تطبيق ساوندبورد لسطح المكتب باستخدام Electron. يسمح لك بتشغيل الأصوات على السماعات وتحويلها في نفس الوقت إلى كابل صوت افتراضي — عشان تسمعها أنت والجماعة في شات ديسكورد.

### ✨ المميزات

| الميزة | الشرح |
|--------|-------|
| **مخرجين صوت** | شغّل الصوت على السماعات والكابل الافتراضي معًا |
| **اختصارات عالمية** | خصص اختصار لوحة مفاتيح لأي صوت |
| **قائمة تشغيل** | سو قائمة والأصوات تشتغل تلقائيًا واحد ورا الثاني |
| **المفضلة** | نجمة على الأصوات اللي تستخدمها كثير وفلترة حسبها |
| **تحديد متعدد** | اختر عدة أصوات — شغّل، احذف، انقل، أو امسح الاختصارات دفعة واحدة |
| **مجلدات** | نظم الأصوات في مجلدات — أنشئ، سمي، احذف |
| **سحب وإفلات** | ارفع ملفات مباشرة أو اسحب البطاقات بين المجلدات |
| **التحكم بالصوت** | مستوى صوت لكل صوت ولكل سيرفر ولكل الأصوات |
| **بحث** | ابحث بالاسم أو استخدم `hk:` عشان تفلتر بالاختصار |
| **ترتيب** | بالاسم، الأحدث، الأقدم، أو حسب الاختصار |
| **عداد تشغيل** | تتبع كم مرة شغلت كل صوت |
| **تنقل بلوحة المفاتيح** | أسهم، Home/End، Enter/Space |
| **كثافة العرض** | مضغوط، وسطي، واسع |
| **ثيم داكن / فاتح** | Catppuccin Mocha للداكن و Latte للفاتح |
| **تحديث تلقائي** | يفحص GitHub عن الإصدارات الجديدة |
| **تصغير للصينية** | شغّل في الخلفية مع قائمة في صينية النظام |

### 🚀 التثبيت

- [VB-Cable](https://vb-audio.com/Cable/) أو أي كابل صوت افتراضي (لتوجيه المايك)

